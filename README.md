### Tools
  - Java 11
  - Intellij IDEA (optional)
  - Selenium WebDriver
  - Gson for Test Data
  - Log4j2
  - Extentreports
  - TestNG for Run Test

### Test architecture
  - ScreenPlay pattern
  - Factory pattern for Web drivers
  - Singleton pattern for Json Test Data
  - Test Data Using Model Classes
  - Support Chrome and Firefox 
  - Use Selenium Grid
  - Run Local and Remote
  - BaseTest Class
  - Parallel execution
  - Azure Pipeline
